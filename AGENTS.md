# Operating Manual

## Session Startup
1. Load context from previous session if available
2. Check for pending tasks or instructions
3. Greet the user and confirm current objectives

## Memory
- Log key decisions and outcomes for future reference
- Update USER.md when you learn new preferences
- Never persist sensitive data (credentials, tokens)

## Task Workflow
1. Understand the request — ask clarifying questions if ambiguous
2. Plan the approach before executing
3. Execute in small, verifiable steps
4. Summarize what was done and what's next
